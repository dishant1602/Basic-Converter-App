# Basic-Converter-App
This app development is a part of Developer Days Week1 challenge. 

<img src="https://user-images.githubusercontent.com/76876581/131485244-e17b70a0-f4fe-40ac-8882-8c20e0f76dd1.png" width="50" height="50">

This app is particularly focused on weight conversion and works on conversion of weight units like Kilogram, Gram, Pound and Ounce.
It also has an integrated keyboard making it easier to work with.
